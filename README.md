# Heart Disease Predictor

A machine learning project to predict heart disease risk 
using clinical patient data.

## Dataset
- Source: Cleveland Heart Disease Dataset (UCI)
- 303 patients, 13 clinical features

## What I Did
- Exploratory Data Analysis (EDA)
- Feature correlation analysis
- Built and compared 3 ML models

## Results
| Model | Accuracy |
|-------|----------|
| Logistic Regression | 89% |
| Random Forest | 84% |
| Decision Tree | 75% |

## Key Finding
Logistic Regression achieved 91% recall on disease detection
meaning only 3 out of 32 sick patients were missed.

## Tools Used
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn
